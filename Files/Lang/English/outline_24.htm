Bitmap                                                                                                          Object

BITMAP objects are objects on the form where bitmap images can be displayed. The actual bitmap to be displayed is set at design time. A bitmap image (stored in the Bitmaps folder in the Project Explorer) can only be using in a single Bitmap object.

 

Properties Supported (Set at design time)

Resource ID, Visible

 

Methods Supported (See "Methods")

Hide, ID, Show, Index, Type, Left, Top
